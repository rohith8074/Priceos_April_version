# Collection: `marketevents`

**Topic:** External Demand Signals
**Agent Access:** READ / WRITE (Market Research Agent)

## Description
This collection stores real-world events that impact short-term rental demand. These events are either auto-seeded during onboarding or generated by the Market Research Agent via real-time web search.

## Schema & Fields

| Field | Type | Semantic Meaning |
|---|---|---|
| `orgId` | ObjectId | The owner of this event registry. |
| `name` | String | Event name (e.g., "GITEX Global 2026", "Eid Al Fitr"). |
| `startDate` | String | Opening day (YYYY-MM-DD). |
| `endDate` | String | Closing day (YYYY-MM-DD). |
| `impactLevel` | Enum | `critical`, `high`, `moderate`, `low`. |
| `upliftPct` | Number | The suggested price increase percentage (e.g., 40 for a 40% increase). |
| `isActive` | Boolean | Whether the event should be considered by pricing agents. |
| `description` | String | AI-generated summary of the event's local demand influence. |

## Relationship Logic
- **Pricing Input:** The `Property Analyst` queries this collection for events overlapping its 30-day window and applies the `upliftPct` to its calculations.
- **Dynamic Updates:** The `Market Research Agent` periodically updates this collection with new events found via its inference tools.
